Performance report¶

Introduction¶

All the numbers given in this report are expressed in number of cycles per output pixel, and have been multipled by the number of cores to normalize them. Thus to obtain the total number of cycles, the reported number must be multiplied by the number of output pixels and divided by the number of cores.

For each kernel, the following tables are reported:

  • Per architecture. This gives the number of cycles for each supported architecture for a fixed window size and number of cores.
  • Per window. This gives the number of cycles for several window size, for a fixed architecture and number of cores.
  • Per number of cores. This gives the number of cycles for several number of cores, for a fixed architecture and window size.
