2D convolutions

The common parameters are displayed hereafter. Other specific parameters are documented with the functions. Only the input width and height have to be specified. The output width and height are equal to the input ones minus the width and height of the coefficient matrix minus 1.

Parameters
  • in: Input matrix
  • coeffs: Coefficient matrix
  • out: Output matrix
  • width: Width of the input matrix
  • height: Height of the input matrix
  • inputStride: Stride for the input matrix (in number of elements). Everytime a number of elements corresponding to the width has been processed, the input matrix pointer is incremented by this number of elements
  • outputStride: Stride for the output matrix (in number of elements). Everytime a number of elements corresponding to the output width has been processed, the output matrix pointer is incremented by this number of elements

Signed 8 bits 5x5 2D convolution

Parameters

  • NormShift: Each output pixel is shifted right by this number after it has been multiplied by Normfactor
